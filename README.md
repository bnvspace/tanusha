# Образовательный портал на PHP

Проект работает на `PHP + MariaDB/MySQL`.

## Быстрый старт

1. Поднимите MariaDB/MySQL и создайте базу данных.
2. При необходимости поправьте локальные параметры в `.env` или используйте `.env.example` как шаблон.
3. Запустите встроенный сервер PHP:

```bash
php -S localhost:8000
```

4. Откройте в браузере: `http://localhost:8000/index.php`

Если у вас Apache или Nginx, направьте Document Root на папку проекта. `config.php` автоматически подхватит `.env`, если файл лежит в корне проекта.

## Что происходит при первом запуске

- `config.php` подключает `bootstrap.php`
- приложение создаёт таблицы автоматически, если их ещё нет
- если база пустая, автоматически создаются курс, недели и стандартные пользователи
- старые Flask/Werkzeug-хеши для `admin`, `teacher`, `student1` автоматически переводятся в формат PHP

## Учётные записи по умолчанию

| Роль | Логин | Пароль |
|---|---|---|
| Администратор | `admin` | `admin123` |
| Преподаватель | `teacher` | `teacher123` |
| Студент | `student1` | `student123` |

## Структура проекта

```text
tanusha/
├── bootstrap.php        # создание схемы и стартовых данных
├── config.php           # подключение к БД и запуск bootstrap
├── auth.php             # сессии, авторизация, CSRF helper-ы
├── index.php            # главный роутер PHP-приложения
├── pages/               # PHP-страницы портала
├── static/              # стили и статика
├── uploads/             # загружаемые файлы
├── storage/             # служебная директория приложения
├── lang.php             # переводы
└── portal.db            # legacy SQLite-артефакт старой версии
```

## Примечание по legacy-данным

`portal.db` остаётся в репозитории как технический хвост старой версии. Текущий runtime использует MariaDB/MySQL и стартует через `index.php`.

## Smoke-проверка

После запуска приложения можно прогнать базовую проверку:

```bash
php scripts/smoke.php --base-url=http://127.0.0.1:8000
```

Скрипт создаёт временные тестовые записи, проверяет основные сценарии и затем убирает их за собой.
